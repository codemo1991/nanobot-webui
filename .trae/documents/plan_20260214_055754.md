## Claude Code CLI 集成计划

### 新增文件
1. `nanobot/agent/tools/claude_code.py` - ClaudeCodeTool 实现
2. `nanobot/claude_code/manager.py` - 进程管理器
3. `nanobot/claude_code/watcher.py` - 结果文件监听器
4. `nanobot/claude_code/hook.py` - Hook 脚本生成
5. `nanobot/hooks/nanobot-claude-hook.py` - Stop Hook 脚本
6. `nanobot/skills/claude-code/SKILL.md` - Skill 文档

### 修改文件
1. `nanobot/agent/loop.py` - 注册 ClaudeCodeTool
2. `nanobot/config/schema.py` - 添加配置项
3. `pyproject.toml` - 添加 watchdog 依赖

### 核心机制
- **Hook 推送**: Claude Code 完成时自动触发 Hook，写入结果文件
- **文件监听**: watchdog 监听结果目录，零轮询开销
- **MessageBus 通知**: 通过现有消息总线通知主 agent

- 核心代码: ~500 行
