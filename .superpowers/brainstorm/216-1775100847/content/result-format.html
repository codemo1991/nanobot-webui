结果展示方式

工具执行完成后，结果以什么形式呈现？

A

纯文本（当前）

JSON 字符串直接渲染，截断时显示 "…"

{
  "status": "ok",
  "data": [...]
}
B

智能格式化

自动识别 JSON/文件列表/命令输出，渲染成可读格式

status: ✓ ok
data: 3 项
  ├─ src/app.py (2.1KB)
  ├─ src/main.py (1.3KB)
  └─ README.md (0.5KB)
C

可折叠详情 + 摘要

结果先显示摘要（如"读取了 3 个文件，共 150 行"），点开查看完整内容

结果摘要:
  ├─ 3 个文件
  ├─ 150 行代码
  └─ 包含 2 处 TODO

展开查看完整内容 ▶
