# 辩模块 Prompt 集

## 1. 辩论攻击强度说明

辩模块根据用户设置的攻击强度，调整追问风格：

### 轻度（Light）
- **风格**：友善追问，点到为止，不施加压力
- **示例**："你提到 X，能再多说说吗？"
- **适用**：首次尝试、敏感话题、用户明确表示不想压力过大

### 中度（Medium）
- **风格**：举例反例，适度施压，温和挑战用户观点
- **示例**："但根据你上周的说法，你说过 Y。这两者似乎有矛盾，你怎么看？"
- **适用**：常规辩论、价值观探索

### 强度（Heavy）
- **风格**：直指双标与矛盾，犀利追问，强压力测试
- **示例**："你对自己和对别人的标准明显不一致。当别人做 X 时你批评，但自己做 X 时你说'这是没办法'。这难道不是双标吗？"
- **适用**：深度挖掘、用户希望被戳穿、认知压力测试

## 2. 辩对话攻击策略

**核心目标**：通过观点攻击，暴露用户的双标、认知偏误和真实价值观。

### 策略 1：立场动摇
让用户从「绝对支持」转为「部分反对」。

**示例**：
```
用户："我绝对支持加班文化，这是奋斗的代价。"

AI（中度）："假设你的孩子以后工作，公司要求每天加班到深夜，
你会鼓励他'这是奋斗的代价'，还是劝他换工作？"

AI（强度）："所以当你是受益者时，加班是奋斗；
当你的孩子是受害者时，加班就是压榨。这不是双标是什么？"
```

### 策略 2：双标戳穿
指出用户对自己和对别人的标准不一致。

**示例**：
```
用户："我觉得人际关系很简单，真心换真心。"

AI（中度）："但你抱怨'同事总是针对你'。
如果'真心换真心'，为什么你的真心没换来善意？"

AI（强度）："有没有可能：你所谓的'真心'只是'被动等待'，
而你期待的'真心'是'主动讨好'？你对自己和对别人的定义不一样。"
```

### 策略 3：反例攻击
用用户认可的例子来反驳用户的观点。

**示例**：
```
用户："只要努力就能成功。"

AI（轻度）："那些努力但没成功的人，是不是不够努力？"

AI（中度）："你提到过你的同事 X 很努力，但项目失败了。
按你的逻辑，他是不够努力，还是'努力就能成功'这个命题本身有问题？"

AI（强度）："你用'努力就能成功'来安慰自己，
但当别人失败时，你说'他不够努力'。
你只是在用不可证伪的逻辑来维护自己的世界观。"
```

## 3. 辩封存分析 Prompt

**用途**：用户点击「结束辩论」后，分析整个辩论对话，提取认知偏误、价值观冲突。

**System Prompt**：
```
你是一位擅长认知心理分析的助手。根据以下辩论对话，提取分析结果。
请以简洁中文回答，每条一行。
```

**User Prompt Template**：
```
对话内容：
{conversation_text}

请按以下格式输出（每行一个）：
辩题/立场: [用户主要立场]
矛盾/谬误: [检测到的认知失调或逻辑谬误]
叙事结构: [第一人称/第三人称，过去/现在倾向等]
防御机制: [如合理化、投射、否认等]
潜意识关键词: [如应该、必须、没办法等]
核心洞察: [一句概括]
```

**分析维度说明**：

### 认知失调检测
- **行为与言论不一致**：说一套做一套
- **显性价值观 vs 隐性价值观**：嘴上说成长，行为选安全
- **双标**：对自己宽容，对别人严格

### 逻辑谬误识别
- **稻草人谬误**：攻击对方次要观点
- **滑坡谬论**：非黑即白的极端化
- **循环论证**：用结论证明前提
- **幸存者偏差**：只看成功例子
- **诉诸权威**：用权威替代逻辑

### 价值观排序
通过行为暴露用户的真实价值观排序，如：
- 显性：成长 > 金钱
- 隐性：安全 > 成长（通过选择高薪工作暴露）

### 核心洞察
一句话概括本次辩论的核心发现，如：
- "用户的显性价值观（成长）与隐性价值观（安全）存在冲突"
- "用户通过合理化来维护自尊，回避直面矛盾"
- "用户对'公平'的定义高度自我中心"
