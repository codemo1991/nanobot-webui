# 赏模块 Prompt 集

## 1. 赏命题设计矩阵

**核心理念**：审美偏好无法伪装，直接对应认知功能与原型。

### 命题主题与选项设计

| 主题 | 选项A特征 | 选项B特征 | 测量维度 |
|------|----------|----------|----------|
| 工作驱动力 | 独自建城堡 | 众人工地 | 独立 vs 协作 |
| 情绪表达 | 压抑的蓝色 | 爆发的红色 | 内敛 vs 外显 |
| 未来愿景 | 稳定的城堡 | 远航的帆船 | 安全 vs 冒险 |
| 自我认知 | 完整的圆 | 破碎的镜像 | 整合 vs 分裂 |
| 人际关系 | 独自看星空 | 篝火旁的聚会 | 独处 vs 连接 |
| 内在力量 | 温暖而明亮 | 深沉而内敛 | 外显 vs 内敛 |
| 秩序与混沌 | 几何图形 | 抽象泼墨 | 结构 vs 流动 |

## 2. 赏生成 Prompt（Qwen-Image）

**用途**：根据命题主题，生成风格迥异的 A/B 图描述，调用 Qwen-Image 文生图。

### 图像生成原则
- **风格对立**：一张写实，一张抽象；一张温暖，一张冷峻
- **隐喻明确**：每张图都有清晰的心理隐喻
- **无文字**：图片不含任何文字，避免理性干扰

### Prompt 模板

**命题：工作驱动力**
```
图A描述：
"一个人在荒野中独自建造一座精致的城堡，月光下，城堡轮廓清晰，
远处是无尽的原野。画面孤独而坚定，写实风格，冷色调。"

图B描述：
"一群人在热闹的工地上一起搭建彩色的乐高塔，
阳光明媚，每个人都在欢笑协作。暖色调，印象派风格。"
```

**命题：情绪表达**
```
图A描述：
"一个蓝色的湖泊，表面平静如镜，没有任何波纹，
周围是寂静的森林。极简主义，冷色调，压抑感。"

图B描述：
"一团红色的火焰在黑暗中爆发，火光四溅，能量释放。
抽象表现主义，强烈的对比，爆发感。"
```

## 3. 赏归因追问 Prompt

**用途**：用户选择 A 或 B 后，AI 追问「为什么选这个？」

**追问话术**：
```
你选择了 {choice}。请花10秒钟感受一下，为什么选它？
不需要过度解释，说出第一直觉就好。
```

**追问原则**：
- 不要诱导答案
- 不要提供选项（「是因为XX吗？」）
- 让用户自由表达，捕捉原始归因

## 4. 赏分析 Prompt

**用途**：结合用户选择与归因，进行荣格类型、大五人格、原型分析。

**System Prompt**：
```
你是一位心理分析师，擅长通过审美偏好进行人格推断。
请根据用户的图像选择与归因，分析其荣格类型、大五人格倾向、原型特征。
```

**User Prompt Template**：
```
命题主题：{topic}
用户选择：{choice}（图A 或 图B）
图A特征：{description_A}
图B特征：{description_B}
用户归因：{attribution}

请按以下格式输出分析：

1. 荣格类型推断
   - 认知功能：[如 Intuition(N) + Thinking(T)]
   - 类型代码：[如 NT型]
   - 解释：[一句话说明为什么]

2. 大五人格线索
   - 开放性(O)：[高/中/低 + 证据]
   - 外向性(E)：[高/中/低 + 证据]
   - 其他维度：[待定/需更多数据]

3. 荣格原型识别
   - 主原型：[如 隐士 The Hermit]
   - 次原型：[如 智者 The Sage]
   - 核心恐惧：[如 被控制/被卷入无效社交]
   - 核心需求：[如 掌控感 Mastery]

4. 与悟/辩交叉验证
   - 是否与悟的叙事模式一致：[是/否 + 原因]
   - 是否与辩的价值观一致：[是/否 + 原因]
```

## 5. 赏艺术偏好与人格对应表

### 荣格类型与艺术偏好

| 认知功能 | 艺术偏好 | 图像特征 |
|---------|---------|---------|
| 感觉(S) | 写实、细节丰富 | 写实风景、人物特写、摄影 |
| 直觉(N) | 抽象、隐喻 | 抽象画、概念艺术、超现实主义 |
| 思维(T) | 结构、秩序 | 几何图形、建筑、极简主义 |
| 情感(F) | 温暖、人文 | 温馨场景、印象派、插画 |

### 大五人格与艺术偏好

| 人格维度 | 高分偏好 | 低分偏好 |
|---------|---------|---------|
| 开放性(O) | 现代艺术、抽象画、达利 | 传统风景、写实人物 |
| 宜人性(A) | 和谐、柔和、暖色调 | 冲突、对比、冷色调 |
| 神经质(N) | 焦虑、孤独、深色调 | 平静、和谐、明亮色调 |
| 外向性(E) | 热闹、群体、鲜明色彩 | 独处、安静、柔和色调 |
| 尽责性(C) | 秩序、对称、结构化 | 随意、不对称、流动感 |

### 荣格原型与图像特征

| 原型 | 图像特征 | 心理含义 |
|-----|---------|---------|
| 英雄 | 战斗、征服、冒险 | 追求成就，害怕失败 |
| 智者 | 沉思、书籍、探索 | 追求真相，恐惧无知 |
| 隐士 | 独自、山间、冥想 | 追求内在，逃避社交 |
| 叛逆者 | 破坏、革命、反叛 | 追求自由，恐惧被控制 |
| 照顾者 | 温暖、拥抱、滋养 | 追求连接，恐惧孤独 |
| 创造者 | 建造、创作、表达 | 追求意义，恐惧平庸 |
